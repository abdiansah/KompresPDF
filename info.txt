=== TENTANG KOMPRES PDF

Kompresi PDF adalah aplikasi untuk melakukan kompresi file pdf menggunakan
bantuan pustaka ghostscript (http://web.mit.edu/ghostscript/www/Ps2pdf.htm).
Aplikasi ini dibuat menggunakan bahasa pemrograman Python dengan pustaka antarmuka
QT5. Konversi bisa untuk satu atau lebih file pdf dengan beberapa pilihan jenis
kompresi.


=== PETUNJUK PENGGUNAAN SINGKAT

1. Pilih lokasi file/folder, pilih "file" jika hanya satu file pdf yang akan
   dikompres, pilih "folder" jika banyak file pdf yang akan dikompres
2. Pilih jenis kompresi, ada empat jenis kompresi yang disertai dengan informasi
   kualitas kompresinya
3. Tekan tombol [Kompres] untuk memulai kompresi
4. Informasi hasil kompresi akan ditampilkan
5. Tekan tombol [Folder] untuk menuju ke lokasi file hasil kompresi
6. Tekan tombol [Keluar] untuk keluar dari aplikasi
7. Tekan tombol [Manual] untuk informasi lebih lanjut


=== PERINGATAN

Aplikasi Kompresi PDF hanyalah antarmuka dari pustaka ghostscript. Kualitas
hasil kompresi tergantung dari kinerja pustaka tersebut. Sejauh pengujian yang
dilakukan, hasil kompresi cukup memuaskan.
